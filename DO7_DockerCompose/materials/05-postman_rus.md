# Инструкция по запуску тестов

1. Установить и запустить **Postman**.
2. Импортировать коллекцию `src/application_tests.postman_collection.json`.
3. При необходимости адаптировать коллекцию для работы с развернутым окружением.
4. Нажать кнопку **Run** в верхнем правом углу.
5. Удостовериться, что все тесты прошли успешно.
